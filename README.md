# Nova IA - Frontend Web 🚀

Repositorio oficial del sitio web B2B de **Nova IA**, agencia especializada en integración de Agentes de IA autónomos y Desarrollo Web de alta conversión en Colombia.

## 🧠 Filosofía de la Marca
**"Tecnología con Rostro, Resultados con Datos"**
El diseño de este sitio prioriza el Retorno de Inversión (ROI), la velocidad de carga y la eliminación de fricción para la captación de leads empresariales vía WhatsApp.

## 🛠️ Stack Tecnológico Actual
- **Estructura:** HTML5 Semántico
- **Estilos:** Tailwind CSS (vía CDN)
- **UI/UX:** Componentes inspirados en Glassmorphism y Claymorphism (B2B Tech Premium)
- **Hosting:** GitHub Pages

## 🎨 Paleta de Colores (Brand)
- **Fondo Principal:** Neutral 950 (`#0a0a0a`)
- **Acento Principal (Brand 500):** Blue 600 (`#2563eb`) - Usado para CTAs primarios.
- **Tipografía:** Inter (Sans-serif geométrica)
